Optimisation SEO en injectant automatiquement les balises meta Open Graph et Twitter Cards à vos articles.

== Description ==

Injecte automatiquement à tous vos articles les balises meta Open Graph et Twitter Cards :

* og:type
* og:title
* og:description
* og:url
* og:site_name
* og:image
* twitter:card
* twitter:domain
* twitter:image:src

== Installation ==

1. Uploadez le dossier `seo-meta` dans `/wp-content/plugins/`
2. Activez le plugin depuis le menu `Extensions` de Wordpress
